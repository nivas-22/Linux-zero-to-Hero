# 30-Day Linux Learning Roadmap

A structured plan to go from **Linux beginner → confident user** in 4 weeks.

---

# Week 1: Linux Foundation (Zero → Confident)

### Day 1: Introduction to Linux
- What is Linux
- Linux vs Windows
- Linux distributions
- Basic commands

### Day 2: Linux File System Structure
- Root directory `/`
- Important directories (`/home`, `/etc`, `/var`, `/usr`)
- Understanding Linux hierarchy

### Day 3: Directory Operations
- `pwd`
- `ls`
- `cd`
- `tree`

### Day 4: File Operations
- `touch`
- `cp`
- `mv`
- `rm`
- `stat`

### Day 5: File Viewing Commands
- `cat`
- `tac`
- `more`
- `less`
- `head`
- `tail`

### Day 6: Text Editors
- `nano`
- `vim`
- Editing files from terminal

### Day 7: Week 1 Review & Practice
- Practice commands
- Create files & directories
- Navigate file system

---

# Week 2: Text Processing & Permissions

### Day 8: Text Processing
- `grep`
- `cut`
- `sort`

### Day 9: Advanced Text Processing
- `awk`
- `sed`
- `wc`

### Day 10: File Permissions Basics
- Read, Write, Execute
- Permission format (`rwx`)

### Day 11: Permission Commands
- `chmod`
- `chown`
- `chgrp`

### Day 12: Special Permissions & umask
- SUID
- SGID
- Sticky bit
- `umask`

### Day 13: Links
- Hard links
- Soft (symbolic) links
- `ln`
- `ln -s`

### Day 14: Week 2 Review & Practice
- Text processing exercises
- Permission examples

---

### Day 15: User Management

### Day 16: Group Management
- `groupadd`
- `groupdel`
- `groups`

### Day 17: sudo & Root Access
- `sudo`
- Root privileges

### Day 18: Process Management
- `ps`
- `top`
- `htop`

### Day 19: Job Control & Background Processes
- `&`
- `jobs`
- `bg`
- `fg`
- `kill`

### Day 20: System Monitoring
- `free`
- `uptime`
- `df`
- `du`

### Day 21: Week 3 Review & Practice
- User & process management exercises

---

# Week 4: Networking, Services & Advanced

### Day 22: Networking Basics
- `ping`
- `ip`
- `ss`
- `netstat`

### Day 23: SSH & Remote Access
- `ssh`
- `scp`
- `sftp`

### Day 24: Package Management
- `apt`
- `yum`
- `dnf`

### Day 25: Services & Systemd
- `systemctl`
- Start / Stop services
- Enable / Disable services

### Day 26: Disk & Storage Management
- `lsblk`
- `mount`
- `df`
- `du`

### Day 27: Compression & Archives
- `tar`
- `gzip`
- `zip`
- `unzip`

### Day 28: Shell Scripting Basics
- Bash scripts
- Variables
- Basic automation

### Day 29: Cron Jobs & Scheduling
- `cron`
- `crontab`
- Task automation

### Day 30: Final Review & Real-World Practice
- Combine all commands
- Mini Linux projects
- Practice system tasks

---

# Goal

By the end of 30 days you should be able to:

- Navigate Linux confidently
- Manage files and directories
- Handle users and permissions
- Monitor processes and system resources
- Work with networking and services
- Automate tasks using shell scripts
