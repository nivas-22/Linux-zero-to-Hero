# 🧪 Hands-On Practice

## 🎯 Practice Exercises

Try the following commands to build practical skills in remote access and network tools:

---

### 🔑 1. Generate SSH Key

```bash
ssh-keygen -t ed25519
```

### 📥 2. Download File with wget
```
wget https://www.google.com/robots.txt
```
- 🌐 Downloads a file from the web
- 📄 Saves it in the current directory
- ⚡ Simple and non-interactive

### 🌐 3. API Request with curl (Headers Only)

```
curl -I https://www.google.com
```

- 📡 Fetches only HTTP response headers
- 🔍 Useful for checking status codes and metadata
- 🚀 Quick way to inspect server responses

### 🔗 4. Test REST API

```
curl https://httpbin.org/get
```
- 🌍 Sends a GET request to a test API endpoint
- 📊 Returns JSON response with request details
- 🧪 Commonly used for API testing and learning
