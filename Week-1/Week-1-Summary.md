# Linux Basic Commands Cheat Sheet

| Command | Purpose | Common Usage |
|---|---|---|
| `whoami` | Current user | `whoami` |
| `pwd` | Current directory | `pwd -P` |
| `date` | Show or set date | `date +"%Y-%m-%d"` |
| `uname` | System information | `uname -a`, `uname -r` |
| `hostname` | System hostname | `hostname -I` |
| `ls` | List files | `ls -la`, `ls -lh` |
| `tree` | Directory tree | `tree -L 2` |
| `mkdir` | Create directory | `mkdir -p path/to/dir` |
| `rmdir` | Remove empty directory | `rmdir -p` |
| `cd` | Change directory | `cd -`, `cd ~` |
| `stat` | File status information | `stat file.txt` |
| `touch` | Create or update file timestamp | `touch file.txt` |
| `cp` | Copy files | `cp -rp src/ dest/` |
| `mv` | Move or rename files | `mv -v old new` |
| `rm` | Remove files or directories | `rm -ri folder/` |
| `find` | Search files in directories | `find . -name "*.txt"` |
| `locate` | Fast file search using index | `locate -i file` |
| `cat` | Display file contents | `cat -n file.txt` |
| `tac` | Display file in reverse order | `tac file.txt` |
| `more` | View file page by page | `more file.txt` |
| `less` | Advanced file pager | `less -N file.txt` |
| `head` | Show first lines of file | `head -20 file.txt` |
| `tail` | Show last lines of file | `tail -f logfile` |
