# 🧪 Linux Hands-On Practice  
## 📅 Day 1 Practice Exercises

---

## 🔹 Exercise 1 — Find Your Username

**Command**

```bash
whoami
```

**Description**

Displays the currently logged-in username.

---

## 🔹 Exercise 2 — Display Current Directory

**Command**

```bash
pwd
```

**Description**

Shows the present working directory.

---

## 🔹 Exercise 3 — Show Date in YYYY-MM-DD Format

**Command**

```bash
date +"%Y-%m-%d"
```

**Description**

Prints the current date in `YYYY-MM-DD` format.

---

## 🔹 Exercise 4 — Display All System Information

**Command**

```bash
uname -a
```

**Description**

Shows detailed system information including kernel name, version, architecture, and more.

---

## 🔹 Exercise 5 — Find Your Kernel Version

**Command**

```bash
uname -r
```

**Description**

Displays the current Linux kernel version.

---

## 🔹 Exercise 6 — Display Hostname and IP Address

**Commands**

```bash
hostname
```

```bash
hostname -I
```

**Description**

- `hostname` → Shows the system hostname  
- `hostname -I` → Displays the IP address assigned to the system

---

## 🔹 Exercise 7 — What Date Will It Be in 7 Days?

**Command**

```bash
date -d "+7 days"
```

**Description**

Calculates and displays the date **7 days from today**.

---

✨ **Tip:**  
Run each command directly in your Linux terminal to see the output.
